# Modo de uso

## Estructura de carpetas

![Folder structure](img/folder-structure.png)

Todo el contenido de la página irá dentro de la carpeta components. 

Dentro de esta carpeta está la carpeta common, necesaria para el correcto funcionamiento de la página. Dentro de esta carpeta el único archivo que hay que modificar el el archivo components/common/assets/constants.json.

Al mismo nivel que la carpeta common, se encuentran las carpetas que representarán el contenido de cada uno de los enlaces que aparecen en el navbar:

![Navbar](img/navbar.png)

(En este caso el enlace de Main Topic equivale a la carpeta de main-view y el de Another Topic equivale a la carpeta another-view)
